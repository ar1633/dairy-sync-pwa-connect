üñ•Ô∏è For System Administrators - Server Setup

Step 1: Install CouchDB (Optional - for LAN sync)

Ubuntu/Debian:

sudo apt update
sudo apt install couchdb
sudo systemctl start couchdb
sudo systemctl enable couchdb

Windows:

Download CouchDB from https://couchdb.apache.org/

Configure CouchDB:

  • Access CouchDB admin at: http://localhost:5984/_utils
  • Create admin user (admin/password)
  • Enable CORS for web access

Step 2: Deploy the PWA

Option A: Static File Server (Recommended)

# Install a simple HTTP server
npm install -g http-server

# Serve the built files
http-server dist -p 3000 -a 0.0.0.0

Option B: Apache/Nginx

Copy the built files to your web server's document root.

Step 3: Network Configuration

Ensure the following ports are accessible on your LAN:

  • Port 3000: PWA application
  • Port 5984: CouchDB (if used)
